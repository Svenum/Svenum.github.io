Speichersysteme im Überblick

DAS
Das DAS (Direct Attatched Storage)ist das üblichste Speichersystem, das in den meisten Computern verwendet wird. Dabei ist die Festplatte direkt mit dem Computer Verbunden.
Vorteile Nachteile
- -
günstig keine flexibelität
kein Remote zugriff

NAS
Beim NAS (Network Attatched Storage) wrid ein Laufwerk über das Netzwerk mittels eines Protokolls wie z. B: SMB, WebDAV, FTP eingebunden. Die Daten werden dabei Dateiorientiert versendet, da die aufteilung auf die Blocks von dem Server erledigt wird (Dateisystem ist ebenfalls ausgelagert).
Vorteile Nachteile
- -
remoter zugriff abhängig von Netzwerkgeschwindigkeit
belastet das Netzwerk

SAN
Hier werden die Daten ebenfalls auf einen Remoten Speicher abgelegt. Im gegensatz zur NAS, wird beim SAN (Storage Area Network) das Dateisystem NICHT auf den Server ausgelagert. Die Datenübertragung ist hier Blockorientiert. Hier werden Technologien wie FC (Fibre Channel), FCoE (Fibre Channel over Ethernet) oder iSCSI (Internet SCSI)
Vorteile Nachteile
- -
redundant teuer
skalierbar kleinere Reichweite
höherer aufwand

Technologien

Fibre Channel
Frame
Vorteile - kleiner Frame -> schnell
Nachteile - teuer
- benötigt spezielle Hardware
- maximale reichweite von 1,2km

Fibre Channel over Ethernet
Frame
Vorteile - es können normale (Ethernet-)Switche verwendet werden
- günstiger als FC
Nachteile - recht teuer
- maximale reichweite von 1,2km

Internet SCSI
Frame
Vorteile - hohe Reichweite
- normale Netzwerktechnik kann verwendet werden
- günstig
Nachteile - je anch Aufbau belastet restliches Netzwerk
- langsam
