Kalkulation

Vorwärtskalkulation:
Listeneinkaufspreis (brutto)
- Umsatzsteuer
= Listeneinkaufspreis (netto)
- Lieferantenrabatt / Einkaufsrabatt
= Zieleinkaufspreis
- Lieferantenskonto / Einkaufsskonto
= Bareinkaufspreis
+ Bezugskosten
= Bezugspreis (oder Einstandspreis)
+ Handlungskosten
= Selbstkosten
+ Gewinn
= Barverkaufspreis
+ Kundenskonto / Verkaufsskonto
= Zielverkaufspreis
+ Kundenrabatt
= Listenverkaufspreis (netto)
+ Umsatzsteuer
= Listenverkaufspreis (brutto)

-> Man rechnet von Oben nach unten

Differenzkalkulation:

-> Man rechnet von Oben und Unten Richtung Gewinn. Dadurch kann man den Gewinn ausrechnen, wenn der Listenverkaufspreis vom Markt gegeben ist.
