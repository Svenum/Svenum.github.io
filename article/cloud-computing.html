Definition

Cloud Computing heißt nichts anderes als das jemand die Rechenleistung seiner Server anderen über das Internet zur Verfügung stellt.

Bereitstellungsmodelle

Es gibt verschiedene Arten der Cloud, diese Arten werden auch Bereitstellungsmodelle genannt.

Public Cloud

Bei einer Public Cloud ist jeder Berechtigt sie zu nutzen. ein Beispiel ist z. B. YouTube oder DropBox

Private Cloud

Eine Private Cloud ist z. B. das Private Rechenzentrum einer Firma, welches über das Internet erreichbar ist. Also ist auch nur eine bestimmte Personen-Gruppe berechtigt sie zu Nutzen. Nicht jeder beliebige kann diese Cloud nutzen.

Hybrid Cloud

Die Hybrid Cloud verbindet die ersten zwei genannten Cloud Modelle. Das heißt, wenn eine Firma eine öffentliche Cloud wie z. B. Azure benutz, jedoch trotzdem noch On-Preme eine Cloud betreibt ist dies eine Hybrid-Cloud.

Community Cloud

Community-Clouds sind ähnlich wie Private Cloud, nur das mehrere ausgewählte Gruppen auf sie zugreifen dürfen. Ein Beispiel sind z. B. mehrere Unis, die sich eine Cloud Teilen um Forschungsergebnisse aus zu tauschen.

Servicemodelle

Unter den Service Modellen versteht man welchen Service ein Cloud-Provider anbietet. Es gibt drei Formen.

IaaS

Infrastruktur As A Service (IAAS) ist, wenn der Kunde vollen Zugriff auf die Cloud bekommt und sich seine eigene Infrastruktur aufbauen kann. Zugriff auf:

  • Netzwerk
  • Betriebssystem
  • Storage
  • …

Beispiele:

  • Linode
  • Azure
  • Oracle Cloud

PaaS

Bei Platform As A Service erhält der Kunde eine Platform mit Entwicklungstools und Bereitstellungsmöglichkeiten von Programmen (oft Webseiten).

Beispiele:

  • YouTube

SaaS

Software As A Service ist eine Art Lizenz die der Kunde erwirbt. Jedoch erhält er nicht nur die Rechte die Software zu verwenden, sondern sie wird ihm auch direkt über das Internet bereit gestellt.

Beispiele:

  • Office 365
